# 📊 Shipment Data Analysis & Interactive Dashboard

## 📌 Project Overview
This project focuses on analyzing shipment and customer data using Microsoft Excel. The goal is to clean raw data, perform analysis, and build an interactive dashboard to generate meaningful business insights.

---

## 📁 Workbook Structure

### 1. Raw Data
- Contains original shipment dataset  
- Includes fields like:
  - Order ID  
  - Warehouse Block  
  - Mode of Shipment  
  - Customer Rating  
  - Product Cost  
  - Discount Offered  
- Used as the base for data processing  

---

### 2. Cleaned Data
- Removed duplicate entries  
- Handled missing/null values  
- Corrected data types (numeric, text, date)  
- Prepared dataset for analysis  

---

### 3. Data Analysis (Pivot Tables)
- Created pivot tables for summarization  
- Analysis based on:
  - Shipment Mode  
  - Warehouse Block  
  - Customer Ratings  
  - Product Importance  
- Helps identify trends and patterns  

---

### 4. Dashboard
- Interactive Excel dashboard  
- Includes:
  - Bar charts  
  - Pie charts  
  - KPIs (Total Orders, Revenue, Ratings)  
- Filters/slicers for dynamic analysis  

---

### 5. Insights (if included)
- Key business findings:
  - Best shipment method  
  - Customer satisfaction trends  
  - Impact of discounts on sales  
  - Logistics performance insights  

---
